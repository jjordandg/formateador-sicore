Instalación del formateador de Sicore 

1. Copiamos la carpeta Sicore en la raíz del disco C. 
2. Dejamos un acceso directo en el escritorio del archivo SicoreVer3_intGrafica.py en el escritorio. 
3. Instalamos Python en el equipo
4. Instalamos las dependencias de la aplicación 
(tkinter, pandas, openpyxl) 

¿Cómo se instalan las dependencias? 
Una vez instalado Python en nuestro sistema, abrimos una terminal o consola de comandos e ingresamos: 
pip install tkinter 
pip install pandas
pip install openpyxl 

Si todo esto salió bien, la aplicación está lista para usarse. 
Se recomienda cambiar el nombre del acceso directo del escritorio por: 
"Formateador Sicore".
